Funciones del sistema operativo Windows

La función principal que desempeña cualquier Sistema operativo es la de hacer de intermediario entre los elementos físicos que componen una computadora (la pantalla, el teclado, el disco duro, la impresora, etc.) y las personas que usan a la computadora, haciendo así más fácil su manejo (CVA - Centro Virtual de Aprendizaje, s. f.).

El sistema operativo Windows está diseñado para ser un entorno multiusuario en el que varios usuarios pueden utilizar el mismo sistema operativo (EBSCO Research, s. f.).

La conexión a Escritorio remoto permite que varios usuarios utilicen Windows simultáneamente. Windows también permite a los usuarios ajustar la configuración del sistema operativo para varias cuentas, creando esencialmente varias versiones del sistema operativo en el mismo equipo (EBSCO Research, s. f.).

Además, es un sistema multitarea en el que se pueden ejecutar varios procesos a la vez. Windows 10 introdujo un nuevo asistente virtual llamado Cortana con el objetivo de competir con la popularidad de Siri de Apple (EBSCO Research, s. f.).
